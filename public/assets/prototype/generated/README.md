# Prototype Generated PNG Assets

First-pass visual assets generated for the Heart Cabin high-fidelity prototype refresh.

## Main Assets

- `notebook-page-bg.png`: vertical scrapbook notebook background for page-level composition.
- `cabin-cutout.png`: transparent miniature cabin hero/result asset.
- `room-cutaway.png`: transparent interactive play-room diorama asset.
- `room-cutaway-clean.png`: simplified transparent play-room stage for generated clue prop overlays.
- `scrapbook-ui-sheet.png`: transparent sprite sheet containing reusable paper UI pieces.
- `clue-props-sheet.png`: transparent sprite sheet containing clue/decorative props.
